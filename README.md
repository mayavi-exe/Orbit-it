## 🚀 Overview

**CampusConnect** is a **college-exclusive social + dating platform** built to create a trusted campus ecosystem.

Unlike traditional apps, it focuses on:

- 🎓 Verified college-only access  
- 🧑‍🤝‍🧑 Community-first interactions  
- 💬 Real-time communication  
- 🎯 Meaningful connections (not just swiping)  

---

## ✨ Features

### 🧑‍🤝‍🧑 Community Feed
- College-specific posts  
- Anonymous confessions  
- Memes, discussions, events  
- Like & comment system  

---

### 💬 Real-Time Chat
- 1:1 messaging using Socket.io  
- Read receipts  
- Offline message sync  

---

### 🎯 Smart Matching System
- Interest-based recommendations  
- Activity-based ranking  
- Optional swipe interaction  
- Mutual match unlocks chat  

---

### 🔐 Verification System
- College email verification  
- Student ID validation  
- Fake profile prevention  

---

### 🛡️ Safety & Moderation
- Report & block users  
- AI content moderation  
- Admin dashboard  
- Privacy controls  

---

### 📅 Events & Clubs
- Discover campus events  
- Join clubs  
- Meet through activities  

---

## 🧠 Tech Stack

### 📱 Frontend
- React Native (Expo)  
- TypeScript  
- NativeWind (Tailwind CSS)  
- Zustand / Redux Toolkit  

---

### ⚙️ Backend
- Node.js + Express  
- MongoDB (Mongoose)  
- Socket.io (real-time)  

---

### ☁️ Services
- Cloudinary (media storage)  
- Firebase (push notifications)  
- JWT Authentication
